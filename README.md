# Manipulating MT5 Orders using Python
In this repo you'll learn to use some of the main MetaTrader 5 functions, all explained in brazillian portuguese. These functions are:
### Orders
- orders_total()
- orders_get()
### Positions
- positions_total()
- positions_get()
### Historical Data
- history_orders_total()
- history_orders_get()
- history_deals_total()
- history_deals_get()
